HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications HyperText Markup Language, commonly referred to as HTML, is the standard markup language used to create web pages. Along with CSS, and JavaScript, HTML is a cornerstone technology, used by most websites to create visually engaging webpages, user interfaces for web applications, and user interfaces for many mobile applications.[1]

Web browsers can read HTML files and render them into visible or audible web pages.
